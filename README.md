
🏀 NBA Predictor

Predict NBA match outcomes using machine learning and real-time data from the official NBA API.


---

📌 Overview

NBA Predictor is an AI-powered project that leverages XGBoost and real NBA data to forecast game results with exceptional accuracy.


---

✨ Features

Live data fetching via the NBA API

Feature engineering and preprocessing

Trained with XGBoost classifier

Cross-validation for performance evaluation

Test Accuracy: 99%

Ideal for data analysts, sports bettors, and ML enthusiasts

python nba.py

⚙️ Installation

1. Clone the repository:



git clone https://github.com/amirhosssein0/NBA
cd NBA

2. Install dependencies:



pip install -r requirements.txt

3. Run the predictor:



python nba.py
